# Decisions

Use this document for settled project decisions that future contributors might otherwise revisit without knowing the original context. Implementation facts belong in the relevant topical document; this file preserves why a consequential choice was made.

## Decision Record

For each decision, create a clearly named section and record:

- the decision
- the context or problem that required it
- the alternatives considered when relevant
- the rationale and tradeoffs
- whether the decision is settled or provisional
- the conditions that would justify revisiting it

_Replace this template section with project-specific decision records as decisions are made, and update an existing record when its status or revisit conditions change._
