# umbra — Platform SDK & Application Harness

## Overview
An opinionated **Standard Operating Environment (SOE)** and **Platform SDK** built on `@base-ui/react` + Tailwind CSS 4. Encapsulates layout architecture, design system tokens, vertical rhythm rules, and application execution bounds.

Installed from `github.com/JedWag/umbra` via:
```bash
npm install git+https://github.com/JedWag/umbra.git
```
